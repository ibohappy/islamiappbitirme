# 🕌 İslami App - Kapsamlı İslami Yaşam Asistanı

**Modern hayatta İslami değerlerle yaşamayı kolaylaştıran, teknoloji ile manevi hayatı birleştiren kapsamlı mobil uygulama.**

---

## 🌟 Uygulamanın Önemi ve Vizyonu

İslami App, modern Müslümanların günlük hayatlarında dini sorumluluklarını yerine getirmelerini kolaylaştırmak amacıyla geliştirilmiş kapsamlı bir İslami yaşam asistanıdır. Teknolojinin gücünü İslami değerlerle birleştirerek, kullanıcılara:

- **Manevi gelişim** için kişiselleştirilmiş rehberlik
- **Dini bilgi** için güvenilir kaynaklara erişim
- **İbadet hayatı** için akıllı hatırlatma sistemi
- **İslami eğitim** için yapay zeka destekli öğretim

sunar ve böylece hem bireysel hem de toplumsal İslami yaşamı destekler.

---

## 🚀 Ana Özellikler

### 📿 **Namaz Vakitleri & Akıllı Bildirimler**
- Konum tabanlı hassas namaz vakti hesaplama
- Kişiselleştirilebilir bildirim sistemi
- Arka plan görev yönetimi ile kesintisiz takip
- Şehir seçimi ve saat dilimi otomatik ayarlama
- Vakit öncesi hazırlık bildirimleri

### 🧭 **Kıble Bulucu**
- GPS ve magnetometre tabanlı hassas kıble yönü
- Gerçek zamanlı pusula gösterimi
- Görsel ve sesli rehberlik
- Kalibrasyon otomatik düzeltme sistemi

### 🤖 **İmam AI - Akıllı İslami Asistan**
- **Gelişmiş AI Chat Sistemi**: OpenAI destekli kapsamlı soru-cevap
- **Kişiselleştirilmiş Yanıtlar**: Kullanıcı profili (yaş, cinsiyet, mezhep) bazında özelleştirilmiş cevaplar
- **Sınırsız Hafıza**: ChatGPT benzeri konuşma geçmişi ve öğrenme sistemi
- **Mezhep Özel Bilgiler**: Hanefi, Şafi, Maliki, Hanbeli mezhep bilgileri
- **Yaş Uygun İçerik**: Çocuk, genç, yetişkin, yaşlı kategorilerine özel açıklamalar
- **Seviye Bazlı Öğretim**: Başlangıç, orta, ileri seviye İslami eğitim
- **Konuşma Geçmişi**: Önceki sohbetlere erişim ve devam etme
- **Akıllı Özetleme**: Uzun konuşmaların otomatik özetlenmesi

### 📖 **Günlük Sureler & Kur'an**
- Günlük sure önerileri
- Arapça metin, Türkçe meal ve okunuş
- Ses dosyası desteği (gelecek güncellemede)
- İlerleme takibi ve istatistikler

### 🙏 **Namaz Rehberi**
- Adım adım namaz kılma rehberi
- Görsel ve yazılı açıklamalar
- Farklı namaz türleri için özel rehberlik
- Yeni öğrenenler için detaylı anlatım

### 👤 **Kullanıcı Profili & Hesap Yönetimi**
- **Güvenli Kimlik Doğrulama**: Supabase Auth ile e-posta/şifre sistemi
- **Detaylı Profil Yönetimi**: Ad, soyad, yaş, cinsiyet, mezhep, şehir bilgileri
- **Kişiselleştirme**: Dini seviye (başlangıç/orta/ileri) ayarlama
- **Şifre Güvenliği**: Şifre değiştirme ve sıfırlama
- **Aktivite Takibi**: Namaz, sure okuma, AI sohbet istatistikleri

### 🔔 **Akıllı Bildirim Sistemi**
- **Namaz Vakti Bildirimleri**: Ezan vaktinde otomatik hatırlatma
- **Özelleştirilebilir Ayarlar**: Bildirim tipleri ve zamanları
- **Arka Plan Çalışma**: Uygulama kapalıyken bile aktif sistem
- **Expo Notifications**: Modern bildirim altyapısı

## 🛠️ Teknoloji Yığını

### **Frontend & Mobil**
- **React Native 0.79.2** - Cross-platform mobil geliştirme
- **Expo SDK 53** - Rapid prototyping ve deployment
- **TypeScript** - Type-safe kod geliştirme
- **React Navigation 7** - Sayfa geçişleri ve routing

### **UI/UX & Stil**
- **React Native Elements (RNE UI)** - UI component library
- **React Native Vector Icons** - Zengin ikon kütüphanesi
- **Expo Vector Icons** - Ek ikon desteği
- **Custom Theme System** - Tutarlı tasarım dili

### **Backend & Veritabanı**
- **Supabase** - Backend-as-a-Service
  - PostgreSQL veritabanı
  - Real-time subscriptions
  - Authentication & authorization
  - Row Level Security (RLS)

### **Yapay Zeka & AI**
- **OpenAI GPT Models** - İmam AI için doğal dil işleme
- **LangChain** - AI application framework
- **Custom Memory System** - Gelişmiş konuşma hafızası
- **Chain of Thought** - Akıllı reasoning sistemi

### **Sensörler & Konum**
- **Expo Location** - GPS ve konum servisleri
- **Expo Sensors** - Magnetometre ve accelerometer
- **React Native NetInfo** - Ağ durumu takibi

### **Bildirimler & Arka Plan**
- **Expo Notifications** - Push notification sistemi
- **Expo Background Fetch** - Arka plan görev yönetimi
- **Expo Task Manager** - Scheduled task execution

### **Güvenlik & Depolama**
- **Expo Secure Store** - Güvenli veri saklama
- **AsyncStorage** - Lokal veri yönetimi
- **Expo Auth Session** - OAuth ve authentication flows

### **Medya & Dosya İşleme**
- **Expo AV** - Audio/video player (gelecek özellikler için)
- **React Native Document Picker** - Dosya seçimi
- **PDF Parse** - PDF doküman işleme

### **Development & Build Tools**
- **Babel** - JavaScript transpiler
- **Metro** - React Native bundler
- **ESLint & TypeScript** - Kod kalitesi araçları
- **Gradle** - Android build system

## 🚀 Kurulum ve Çalıştırma

### **Ön Gereksinimler**
```bash
Node.js 18+ ve npm/yarn
React Native CLI
Android Studio (Android için)
Xcode (iOS için)
Expo CLI
```

### **1. Projeyi Klonlayın**
```bash
git clone https://github.com/ibohappy/islamiappbitirme
cd islami-app
```

### **2. Bağımlılıkları Yükleyin**
```bash
npm install
# veya
yarn install
```

### **3. Ortam Değişkenlerini Ayarlayın**
```bash
# src/config/env.example.ts dosyasını kopyalayıp env.ts olarak kaydedin
# Supabase ve OpenAI API anahtarlarınızı ekleyin
```

### **4. Uygulamayı Başlatın**

**Geliştirme Modu:**
```bash
npm start
# veya
expo start
```

**Platform Özel Çalıştırma:**
```bash
# iOS Simülatör
npm run ios

# Android Emülatör
npm run android

# Web Tarayıcı
npm run web
```

## 🔧 Konfigürasyon

### **Supabase Kurulumu**
1. [Supabase](https://supabase.com) hesabı oluşturun
2. Yeni proje açın
3. API anahtarlarını `env.ts` dosyasına ekleyin
4. Gerekli tabloları oluşturun (user_profiles, user_settings)

### **OpenAI API Kurulumu**
1. [OpenAI](https://platform.openai.com) hesabı oluşturun
2. API anahtarı alın
3. `env.ts` dosyasına ekleyin

### **Google Services (Android)**
1. Firebase Console'dan `google-services.json` indirin
2. Proje kök dizinine yerleştirin
